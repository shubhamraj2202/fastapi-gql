# fastapi-gql
Example Python project based on FastAPI and GraphQueryLanguage

# Project Setup for Development
- `source setup.sh`

# Start Server
- Run Command: `python3 -m poetry run uvicorn fastapi_gql.app:app --reload`
- Open Browser: `http://localhost:8000/graphql`
